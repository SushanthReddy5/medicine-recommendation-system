# 🏥 MediAssist — AI Healthcare Assistant

A full-stack AI-powered healthcare platform built with Flask, React, and MongoDB.

## Features
- 🔍 AI Symptom Checker (Top 3 predictions with confidence scores)
- 💊 Drug Interaction Checker
- ⏰ Medicine Reminder System
- 🗺️ Nearby Hospital & Pharmacy Locator
- 📊 Health Analytics Dashboard
- 🤖 AI Medical Chatbot (Groq AI)
- 👤 User Authentication with Health Profile
- 🔐 Forgot/Reset Password via Email

## Tech Stack
- **Frontend:** React, Tailwind CSS, Framer Motion
- **Backend:** Flask, Python
- **Database:** MongoDB
- **ML:** Scikit-learn, TF-IDF + Random Forest
- **AI Chatbot:** Groq (Llama 3.1)

## Setup
1. Clone the repo
2. Create `backend/.env` with your API keys
3. Install dependencies and run

## Note
Never commit your `.env` file — it contains API keys.
